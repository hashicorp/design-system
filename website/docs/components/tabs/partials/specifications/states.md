| Element          | Usage                                           |
|------------------|-------------------------------------------------|
| Tab              | Required, two minimum                           |
| Content area     | Required, Tabs should never be used alone       |
| Selected         | Required, one is always selected                |
| Bottom border    | Required                                        |
| Icon             | Optional                                        |
| Text             | Required                                        |
| Count            | Optional                                        |
| Indicator        | Required, displayed on selected Tab             |

## Anatomy

![Anatomy of the badge component](/assets/components/badge/badge-anatomy.png)

| Element          | Usage                                                                |
|------------------|----------------------------------------------------------------------|
| Text             | Required                                                             |
| Color            | Optional                                                             |
| Icon             | Optional, but required on Status badges (Success, Warning, Critical) |
| isIconOnly       | Optional                                                             |
| Size             | Optional                                                             |
| Type             | Optional                                                             |

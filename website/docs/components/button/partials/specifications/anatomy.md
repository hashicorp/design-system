## Anatomy

![Button anatomy](/assets/components/button/button-anatomy.png)

| Element          | Usage                                           |
|------------------|-------------------------------------------------|
| Text*            | Optional                                        |
| Container        | Required                                        |
| Leading Icon*    | Optional                                        |
| Focus Ring       | Focus State Only                                |
| Trailing Icon*   | Optional                                        |

\* One of these must be present.

| Element          | Usage                                           |
|------------------|-------------------------------------------------|
| Icon             | Optional, but recommended                       |
| Title            | Required, if no description Optional, otherwise |
| Description      | Required, if no title Optional, otherwise       |
| Actions          | Optional                                        |
| Dismiss button   | Optional                                        |
| Content          | Required                                        |

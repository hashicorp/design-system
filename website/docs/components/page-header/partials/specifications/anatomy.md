| Element          | Usage                                           |
|------------------|-------------------------------------------------|
| Title | Required, renders the page title or header. |
| Breadcumb | Optional  |
| IconTile | Optional |
| Badges | Optional |
| Subtitle | Optional |
| Description | Optional |
| Custom metadata | Optional |
| Actions | Optional, renders either one, two, or three actions |

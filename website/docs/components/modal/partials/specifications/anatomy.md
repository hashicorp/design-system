## Anatomy

![Anatomy of a modal](/assets/components/modal/modal-anatomy.png)

| Element | Usage |
| ------- | ----- |
| **Header** |
| Title icon | Optional |
| Tagline | Optional |
| Dismiss | Required, all modals are dismissible |
| **Body** |
| Content | Required; either default (simple text) or custom. |
| **Footer** |
| Actions | Required; either one, two, or three |

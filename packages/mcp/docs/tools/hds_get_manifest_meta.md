# `hds_get_manifest_meta`

Returns top-level component manifest metadata.

## Input

No input arguments.

## Output

```json
{
  "generatedAt": "2026-05-12T00:29:04.586Z",
  "componentCount": 70
}
```

## Notes

- Deterministic for a fixed manifest file.
- Useful as a quick freshness check before using other catalog tools.
